Afterframe — Install & Use

Afterframe sends your Blender camera and objects into After Effects — the camera move, lens, and rotation matched frame for frame — as one self-contained script. No plugins, no dependencies.

1. Install (30 seconds)

  1. In Blender: Edit ▸ Preferences ▸ Add-ons ▸ Install…
  2. Pick the Afterframe .zip you downloaded (don't unzip it first).
  3. Tick the checkbox to enable it. The panel appears in the 3D View's N-panel (press N) under the Afterframe tab.

Works in Blender 4.0 and up. Settings live on the window, so it's fine even with linked/BlenderKit scenes.

2. The panel, field by field

FieldWhat it does
World ScalePixels per Blender unit. 100 is a good default (100 px = 1 Blender unit). Bigger = a bigger comp-space scene.
Comp NameNames the After Effects comp. Leave blank to use your scene name.
NullsWhich objects become AE nulls: Selected objects (just what you've picked), All objects, or All empties. Every null carries position and rotation.
OutputWhere the .jsx is written. Leave blank → next to your .blend (or your Desktop if the file isn't saved). Pick a folder and it uses the Comp Name as the filename.
Plate PROOptional. Point at the first frame of your rendered footage — it's imported as the background and conformed to your scene's frame rate automatically (no hand-interpreting to 24/30).
Screen PROOptional. The mesh you want to replace (e.g. a phone screen). Turns on screen replacement.
Screen Comp PROOptional. The name of an existing AE comp to seat on the screen (anchor auto-set to half its size, scaled + oriented). Adapts to whatever you point it at.
Screen Media PROOptional. A file (PNG/MOV) to seat instead of a comp.

3. Export & bring into After Effects

  1. Set your active scene camera and frame range.
  2. Choose Nulls and select what you want exported.
  3. Click Export .jsx for After Effects.
  4. In After Effects: File ▸ Scripts ▸ Run Script File… → pick the .jsx.

It builds a comp at your render resolution and fps, with everything parented to a BLENDER_WORLD null so Blender's origin lands at the comp center.

4. Screen replacement PRO

Two ways — both seat your content on the tracked surface, scaled, oriented, anchored, riding the move:

A. One-step (in Blender)

Set Screen = your screen mesh, and either Screen Comp (an existing AE comp's name) or Screen Media (a file). Export and run the .jsx — the replacement is already seated as a SCREEN_REPLACE layer.

B. Seat Selected (in After Effects, name-agnostic)

If you'd rather keep your content as a live, editable AE comp:

  1. Set Screen = your screen mesh in Blender, leave Comp/Media blank, export, and run the .jsx. It drops an AF_SCREEN null that carries the seating.
  2. In AE, add any content (a comp or footage) and select that layer.
  3. Run AfterframeSeatSelected.jsx (File ▸ Scripts ▸ Run Script File). It seats the selected layer onto AF_SCREEN — anchor set to half its size, scaled, oriented, riding the spin.

5. What the script builds

LayerWhat it is
BLENDER_WORLDA null at the comp center — Blender's origin. Everything is parented to it.
CameraYour Blender camera: position, orientation, and lens (zoom), animated per frame.
Your nullsOne 3D null per exported object — position + rotation, tracking moves and spins.
AF_SCREEN PROThe seating null for screen replacement (used by Seat Selected).
SCREEN_REPLACE PROYour replacement content, seated on the screen (one-step mode).
PLATE PROYour rendered footage, conformed to the scene fps, as the background.

6. Tips & troubleshooting

  • Plate looks wrong / won't show: for the background, render a single-layer EXR or PNG/MOV. Multilayer EXRs need an extra plugin (EXtractoR) to pull a pass — not needed for a plain plate.
  • Replacement content sideways or off: in Seat Selected mode it reads the size off your selected layer — make sure that layer is the content you want, then run it.
  • Nothing exported / "no camera": set an active scene camera (Scene properties) before exporting.
  • The screen mesh: Afterframe finds the screen's front face automatically — you don't need to clean its object axis.
Frame rate: the Plate field conforms your footage to the scene fps on import — set your Blender scene fps to match your footage and it just lines up.
